Dizzy Mouse

Technologies used: C#, Unity, Photoshop

Dizzy Mouse is a 2-players minigame where each one is a little rat. The goal of the game is to get where the cheese is, but the controls changes every 5 seconds, so it is not as easy as it seems to be.

The game was made in about 12-24 hours for the Alicante 2019 GameJam. The theme we were given was “perspective“, so with that in mind we got some pretty cool (and bad) ideas. In the end, we chose to do this one. We were pretty happy with the results.

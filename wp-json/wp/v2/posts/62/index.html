Dizzy Mouse<\/strong> <\/em>is a 2-players minigame where each one is a little rat. The goal<\/strong> of the game is to get where the cheese is<\/strong>, but the controls changes every 5 seconds<\/strong>, so it is not as easy as it seems to be.<\/p>\n\n\n\n

The game was made in about\u00a012-24 hours<\/strong>\u00a0for the\u00a0Alicante 2019 GameJam<\/strong>. The theme we were given was \u201cperspective<\/strong>\u201c, so with that in mind we got some pretty cool (and bad) ideas. In the end, we chose to do this one. We were pretty happy with the results.<\/p>\n\n\n\n
